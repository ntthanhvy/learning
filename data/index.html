Data Wrangling

pandas & NumPy for ETL/ELT interviews: the methodology behind data processing — tables-not-loops thinking, cleaning, groupby, joins, reshaping — taught through the SQL you already know.

How this works: one short lesson per day, light-touch while the Go and Rust weeks run (through ~14 Jul); after that this becomes a main track and the pace deepens. Every lesson pairs the code with the interview answer — what to type and what to say out loud. Practice files are self-checking: run them with uv run --with pandas python3 practice/<file>.py.
