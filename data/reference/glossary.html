Data Wrangling · Reference

Glossary

Every jargon term used in the lessons, in one place. Tiếng Việt column is dev-style — how Vietnamese devs actually say it, not textbook translation.

TermTiếng Việt (dev-style)In software
data wranglingxử lý / làm sạch dữ liệu thô (hay giữ nguyên "wrangle")Cleaning, restructuring, and enriching raw data into a usable shape — the hands-on craft data jobs test in interviews.
ETLETL (giữ nguyên) — biến đổi xong mới nạp vào khoExtract–Transform–Load: pull data from sources, clean/reshape it in a processing step (often Python/pandas), then load the finished result into the destination.
ELTELT (giữ nguyên) — nạp thô vào warehouse trước, biến đổi sauExtract–Load–Transform: raw data lands in the warehouse first and is transformed there, usually with SQL/dbt. The modern default for analytics pipelines.
DataFrameDataFrame (giữ nguyên) — bảng 2 chiều, giống table SQLpandas' central object: a table of named, typed columns with labeled rows. Your SQL table instincts apply to it almost line for line.
SeriesSeries (giữ nguyên) — một cột của DataFrameOne column of a DataFrame: a 1-dimensional labeled array where every value shares one dtype. df["amount"] returns one.
dtypedtype (giữ nguyên) — kiểu dữ liệu của CẢ một cộtThe data type of an entire column (int64, float64, object/string, datetime64…). Per column, not per cell — like a typed column in Postgres.
NumPyNumPy (giữ nguyên) — thư viện mảng nền tảng, pandas chạy trên nóThe foundational library for fast typed arrays in Python. Each DataFrame column is a NumPy array — that's where pandas' speed comes from.
vectorizedvectorized (giữ nguyên) — thao tác cả cột một lượt, không loopApplying an operation to a whole column/array in one call, executed in compiled C, instead of looping over elements in the Python interpreter. The core pandas methodology.
boolean maskmask (hay nói tắt) — mảng True/False để lọc hàngA True/False Series aligned to the rows; indexing a DataFrame with it keeps only the True rows. pandas' WHERE clause: df[df["amount"] > 100].
NaNgiữ nguyên "NaN" — giá trị thiếu, so sánh == luôn ra Falsepandas'/NumPy's marker for missing floating-point data. Detected with isna() — never with ==, since NaN != NaN by IEEE float rules.
NaTgiữ nguyên "NaT" — NaN dành riêng cho cột datetimepandas' missing-value marker specifically for datetime columns — the datetime64 equivalent of NaN.
Indexgiữ nguyên "Index" — nhãn hàng, tách biệt với các cột dữ liệuThe labels attached to a DataFrame's rows, separate from the data columns. .loc looks up by this label; .iloc looks up by raw position regardless of it.
RangeIndexgiữ nguyên "RangeIndex" — index mặc định 0,1,2,3..., không mang ý nghĩa nghiệp vụThe default pandas Index after loading data: 0, 1, 2, 3… — just positions, carrying no business meaning until you set_index a real key.
coerce (errors="coerce")giữ nguyên "coerce" — ép kiểu, lỗi thì thành NaN chứ không crashA conversion mode (pd.to_numeric/pd.to_datetime) that turns unparseable values into NaN/NaT instead of raising — the pandas equivalent of SQL's TRY_CAST.
